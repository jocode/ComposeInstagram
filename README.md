# Compose Instagram

Este es un ejemplo de la creación de una app al estilo de instagram usando Jetpack compose.

## Clean Architecture

Es una meta arquitectura que nos permite organizar mejor el código. No se puede aplicar de una forma estricta,  existen diferentes esquemas para aplicar los conceptos.

- Un caso de uso es como una única acción
- La inyección de las dependencias son técnicas que nos permiten evitar el acoplamiento entre capas, es más fácil realizar los tests y simplifica el desarrollo.
